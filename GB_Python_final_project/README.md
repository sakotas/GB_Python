# Задача

Напишите код, который запускается из командной строки и получает на вход путь до директории на ПК. Соберите информацию о содержимом в виде объектов namedtuple. Каждый объект хранит: имя файла без расширения или название каталога, расширение, если это файл, флаг каталога, название родительского каталога. Написать 3-5 тестов к задаче.

Сдавать дз ссылкой на репозиторий GitHub(проверьте что он не приватный перед отправкой).

# Решение
Для запуска из консоли и проверки:
`python main.py temp_test_directory`
